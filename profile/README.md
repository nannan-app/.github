### What is Murmur?

Murmur is an open-source desktop diary app where you record your life in a chat-style interface — text, images, moods, and long-form articles. Everything is encrypted locally with AES-256-GCM. Your data never leaves your machine.

### Highlights

- **Chat-style journaling** — Messages appear as chat bubbles, natural and effortless
- **Dual-space privacy** — Wrong password opens a decoy space; no one can tell which is real
- **Local-only encryption** — AES-256-GCM + Argon2, zero cloud dependency
- **AI diary summary** — One-click AI analysis of your day
- **Markdown editor** — Typora-style WYSIWYG for longer writing
- **Cross-platform** — macOS, Windows, Linux (built with Tauri v2 + Rust)

### Tech Stack
