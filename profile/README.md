### What is Murmur? / 喃喃是什么？

Murmur is an open-source desktop diary app where you record your life in a chat-style interface — text, images, moods, and long-form articles. Everything is encrypted locally with AES-256-GCM. Your data never leaves your machine.

喃喃是一款开源桌面日记应用，以聊天界面的形式记录生活——文字、图片、心情、长文，全部通过 AES-256-GCM 本地加密，数据永远不会离开你的设备。

### Highlights / 功能亮点

- **Chat-style journaling / 聊天式日记** — Messages appear as chat bubbles, natural and effortless / 消息以气泡形式呈现，自然流畅
- **Dual-space privacy / 双空间隐私** — Wrong password opens a decoy space; no one can tell which is real / 错误密码打开伪装空间，外人无法分辨
- **Local-only encryption / 纯本地加密** — AES-256-GCM + Argon2, zero cloud dependency / 零云端依赖
- **AI diary summary / AI 日记总结** — One-click AI analysis of your day / 一键 AI 分析当日日记
- **Markdown editor / Markdown 编辑器** — Typora-style WYSIWYG for longer writing / 类 Typora 所见即所得长文编辑
- **Cross-platform / 跨平台** — macOS, Windows, Linux (built with Tauri v2 + Rust)
